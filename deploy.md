# Deployment Guide

This portfolio is deployed using GitHub Pages.

## Steps to Deploy

1. Push code to GitHub repository
2. Ensure code is pushed to the main branch
3. Go to repository Settings
4. Navigate to Pages
5. Select branch: main
6. Save settings
